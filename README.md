## 🛠 What this tool does

AI tools use instruction files to follow your specific coding style. If these files contain errors, the AI might act in ways you do not expect. This tool finds those issues. It checks for formatting mistakes, unclear rules, and conflicts in your setup. 

It helps you maintain consistent behavior across different tools like:
- Claude Code
- Gemini CLI
- Cursor
- GitHub Copilot

## 📝 Understanding your results

The tool displays the status of every file it checks. A green checkmark means the file looks good. A yellow warning icon shows a potential problem. A red icon marks an error that needs your attention.

Click on any line with an icon to see details. The tool suggests changes to improve the file. It explains why a rule might cause confusion for an AI. You choose whether to apply these changes. Remember that this tool does not save changes for you. It provides advice on how to improve your text files manually.

## 💡 Common questions

**Does this tool send my data to the internet?**
No. The scan happens entirely on your machine. Your instruction files stay on your hard drive.

**Can I use this for multiple projects?**
Yes. You can switch between different project folders whenever you want. The tool saves your recent folders to make switching easy.

**What happens if I find an error?**
Open your instruction file in a text editor like Notepad. Apply the changes recommended by the diagnostic tool. Save the file and run the scan again to verify the fix.

**Is it safe to run on my PC?**
The tool operates in read-only mode. It never writes, deletes, or modifies your files. It only scans the content to find errors in your AI rules files.
